# Haxx-theme
Visual Studio Code theme like it's 1999

## Installation

1. Install and Launch [Visual Studio Code](https://code.visualstudio.com/)
2. **Manage** > **Extensions** (or `Ctrl+Shift+X`)
3. Search for `haxx`
4. Click **Install**
5. **Manage** > **Color Theme** > **haxx**
6. All set! Welcome to the jungle
